# Moderno Furnish

This is a simple, responsive website showcasing modern furniture. It demonstrates basic web development skills including HTML, CSS, and JavaScript.

## Features

- Clean, modern design
- Responsive layout for all devices
- Product gallery
- Product Search
- Cart
- Order and Payment Page
- Admin Dashboard

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- TailwindCSS
